Description

Feli is a college student who recently took a part-time job as a teacher. In this part-time job, Feli is tasked to teach and improve the grade of $$$N$$$ students. Initially, the $$$i$$$-th student has an academic grade of $$$X_i$$$. To finish her assignment, Feli must make the final academic grade of each student to exactly $$$M$$$.

In a single study session, Feli can teach multiple students at once, and every student who attends the session gets it's academic grade increases by one. If the grade of a student reached $$$M$$$, then the student feels that they has mastered the lesson and will not attend the study session again.

The friendship of the $$$N$$$ students is very close. If in a study session, the $$$L$$$-th student and $$$R$$$-th student attend the session, then every student between $$$L$$$ and $$$R$$$ must come too.

Help Feli by telling her the minimum study session needed to finish the task, so that she can return to her vacation as soon as possible.

Input Format

Input starts with a single integer $$$T$$$, the number of test cases. The next lines contain the description of each test case as explained below.

The first line of each description contains 2 positive integers $$$N$$$ and $$$M$$$, where $$$N$$$ is the number of students in a study session and $$$M$$$ is the grade every student must reach.

The second line contains $$$N$$$ integers $$$X_i$$$, where $$$X_i$$$ is the initial grade of the $$$i$$$-th student.

Output Format

For each test case, output "Case #$$$X$$$: $$$Y$$$", where $$$X$$$ is the case number (starting from 1), and $$$Y$$$ is the minimum study session needed to finish Feli's task.

Constraints

  • $$$1 \le T \le 100$$$.
  • $$$1 \le N \le 10^5$$$.
  • $$$0 \le M \le 10^5$$$.
  • For each $$$i$$$,$$$ 0 \le X_i \le M$$$.

Sample Input

2
5 5
4 3 2 3 4
5 5
4 4 5 4 3

Sample Output

Case #1: 3
Case #2: 3

Explanation

For the second case, we cannot select the first and fifth student as $$$L$$$ and $$$R$$$, because the third student's grade had reached $$$M$$$.
