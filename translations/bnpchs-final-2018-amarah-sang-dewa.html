Description

The nation of Bollyjee has experienced in-fighting between their tribes (civil war) for a long time. There are $$$K$$$ tribes ($$$1 \le K \le 26$$$) clashing. Each tribe is represented with initials from A to Z. The nation of Bollyjee can be represented as a grid with $$$R$$$ rows $$$\times$$$ $$$C$$$ columns. Each cell can be controlled by only one tribe (labeled by that tribe's initials) or not conquered by any tribe (labeled with .).

Two cells that are adjacent to each other (share an edge) are in the same territory if those two cells are also controlled by the same tribe. If cell $$$U$$$ and cell $$$V$$$ are in the same territory and cell $$$V$$$ and cell $$$W$$$ are in the same territory, therefore cell $$$U$$$ and cell $$$W$$$ are also in the same territory.

Each tribe can have more than one territory or even no territory at all. The grid below shows three tribes where tribes A, B, and C have 2, 1, and 3 territories respectively.

A A . . C . C C
A B B . C A . C
B B C B . A . C
. B B B A A A C

Each tribe really cares about safety protocols. Due to that, the following property is abided for all territories: For each pair of different cells within one territory, there exists exactly one shortest route from one cell to the other.

The shortest route is from cell $$$U$$$ to cell $$$V$$$ is defined as a series of movements in four directions (left, up, right, or down) from cell $$$U$$$ to cell $$$V$$$ such that each traversed cell is within one territory and the number of movements used is minimized.

For example, in the grid below, there exists only one shortest route from B1 to B2, which is left - down - down - right - right.

A A . . C . C C
A B B1 . C A . C
B B C B . A . C
. B B B2 A A A C

Watching his denizens continuously waging wars, God Jelix becomes furious. He passed judgment onto his denizens by conceiving $$$Q$$$ plans of destruction. Every destruction plan is independent of each other. In other words, each destruction plan does not affect another.

For each plan of destruction, the following will be done by God Jelix :

  • God Jelix will choose one sub-rectangle from the grid. This sub-rectangle is represented with two cells referring to the top-left cell ($$$r_1, c_1$$$) and the bottom-right cell ($$$r_2, c_2$$$) of the sub-rectangle.
  • God Jelix will then destroy all cells outside the sub-rectangle that was chosen. Therefore all territories controlled by whichever tribe that exists outside of the sub-rectangle will be gone and no longer controlled by any tribe.
  • After completing the destruction plan, God Jelix will then count how many territories are left in the nation of Bollyjee. There could exist two squares that were at first within one territory, but after the destruction, those squares are now separated (no longer within the same territory).
Your task is to determine how many territories are left in the nation of Bollyjee for each plan of destruction. Keep in mind that each plan of destruction is independent of each other.

Input Format

Input starts with an integer $$$T$$$ â€” the number of test cases.

Each test case starts with three integers $$$R$$$, $$$C$$$, and $$$Q$$$ ($$$1 \le R, C \le 500; 1 \le Q \le 10^5$$$). The next $$$R$$$ lines each contains $$$C$$$ characters denoting the label of each cell. Each character is either a . if the cell is not occupied or capital letters from A to Z if it is controlled by a tribe.

Input is then followed by $$$Q$$$ lines. Each line consisting of four space-separated integers $$$r_1$$$, $$$c_1$$$, $$$r_2$$$, and $$$c_2$$$ ($$$1 \le r_1 \le r_2 \le R; 1 \le c_1 \le c_2 \le C$$$).

C++ users are recommended to use fast input/output, such as scanf/printf.

It is guaranteed that the total queries of all test cases do not exceed $$$3 \times 10^5$$$.

Output Format

For each test case, output Kasus #X:, where $$$X$$$ is the case number (starting from $$$1$$$), followed by $$$Q$$$ lines where each line contains an integer referring to the answer for each destruction plan willed by God Jelix.

Sample Input

1
4 8 3
AA..C.CC
ABB.CA.C
BBCB.A.C
.BBBAAAC
1 1 4 8
2 3 4 5
1 4 2 4

Sample Output

Kasus #1:
6
5
0

Explanation

The grid from the sample case is given in the description.

  • For the first destruction plan, there are clearly no destroyed cells, therefore there are 6 territories left over.
  • For the second destruction plan, tribes A, B, and C have 1, 2, and 2 territories respectively.
  • For the third destruction plan, all cells within the sub-rectangle chosen by God Jelix are unoccupied cells, therefore there are no territories left over.
