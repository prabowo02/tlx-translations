Description

Feli is a student at a popular university. One day, Feli is given a group assignment consisting of $$$N$$$ questions. The $$$i$$$-th question can be solved in $$$X_i$$$ amount of time. Because the group assignment has many questions, Feli formed a group of $$$M$$$ people

To complete the group assignment, Feli's group agrees to split work among them. Each member chooses two integers $$$L$$$ and $$$R$$$, and the member has to solve questions from $$$L$$$ to $$$R$$$ inclusive.

With this method, each question can be solved by more than one member. The questions, however, remain at the same difficulty so that the time needed to solve the $$$i$$$-th question is still $$$X_i$$$. The time it takes for a member to solve the questions is equal to the sum of the time it takes to solve each question.

Feli's group wants to minimize the amount of the time they spend on solving the questions. Help Feli's group determine the minimum time they can spend!

Input Format

Input starts with a positive integer $$$T$$$, the number of test cases, and the next lines will be explained below.

The first line consists of two positive integer $$$N$$$ and $$$M$$$ - the number of questions and Feli's friends. The next line consists of $$$N$$$ integer $$$X_i$$$, the time it takes to solve the $$$i$$$-th question.

Output Format

For each test case, output "Case #$$$X$$$: $$$Y$$$". $$$X$$$ represents the case number (starting from 1), and $$$Y$$$ represents the minimum time Feli's group can spend.

Constraint

  • $$$1 ≤ T ≤ 100$$$
  • $$$1 ≤ N,M ≤ 10000$$$
  • $$$1 ≤ L ≤ R ≤ N$$$
  • $$$1 ≤ X_i ≤ 10^7$$$

Sample Input

2
3 2
3 1 1
3 2
3 2 1

Sample Output

Case #1: 3
Case #2: 3
