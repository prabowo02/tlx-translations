Description

Pak Chanek has a word that consists of lowercase alphabets. He got bored and decided to erase some characters. Pak Chanek will start by finding the alphabetically smallest character in that word, then he will erase it. Pak Chanek will do this operation multiple times. Now he wonders, in the $K$-th erasure operation, what character did he erase?

Input Format

The first line consists of two integers $N$ and $K$ â€” the length of the word and the $K$-th erasure operation respectively.

The second line contains a string of length $N$, consisting of only lowercase alphabet characters. This string is Pak Chanek's word.

Output Format

Output a single integer, the character that is erased in the $K$-th erasure operation.

Sample Input 1

5 3
aabcc

Sample Output 1

b

Sample Input 1

5 4
aabcc

Sample Output 1

c

Subtasks

For all subtasks:

  • $1 \le K \le N \le 100\,000$
  • The word consists of only lowercase alphabet characters ('a' - 'z').

Subtask 1 (22 points)

  • $1 \le K \le N \le 1\,000$

Subtask 2 (23 points)

  • The word consists of only the letters 'a', 'b', or 'c'

Subtask 3 (55 points)
