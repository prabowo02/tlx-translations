Description

The city Depok can be represented as a grid of $N$ rows and $M$ columns. The top-left square of the grid is represented as $(1, 1)$ and the bottom-right square of the grid is represented as $(N, M)$.

Pak Chanek starts at square $(1, 1)$ at the $1$st second. Pak Chanek will then take a stroll around Depok. Each second, he will decide to walk to the square on his right, the square below him, or stay still on the current square he's in.

Now Pak Chanek wonders, how many possible squares that he could be in after $X$ seconds?

Input Format

The first line consists of an integer $T$ — the number of test cases. Each test case will be stated as follows.

A line consisting of three integers, $N$, $M$, and $X$ — the grid size and the number of seconds Pak Chanek has to stroll around respectively.

Output Format

For each test case, output an integer stating the number of possible squares that he could be in after $X$ seconds

Sample Input

2
5 5 2
3 4 10

Sample Output

3
12

Constraints

  • $1 \le T \le 10$
  • $1 \le N, M \le 1\,000\,000\,000$
  • $1 \le X \le 2\,000\,000\,000$
