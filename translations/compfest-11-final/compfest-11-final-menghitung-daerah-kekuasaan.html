Description

Pak Chanek just arrived in an uninhabited island. This island is an $N \times M$ rectangle. The top left corner is square $(1, 1)$ and the bottom right corner is square $(N, M)$.

Because it is an uninhabited island, Pak Chanek wants to develop territories on that island. At first, Pak Chanek has no territories in that island.

Pak Chanek will then perform $Q$ operations of three types, which are:

  • Operation type $1$, Pak Chanek will develop a territory on all squares that are located on row $X$. It is guaranteed that there weren't any type $1$ operation in row $X$ before.
  • Operation type $2$, Pak Chanek will develop a territory on all squares that are located on column $X$. It is guaranteed that there weren't any type $2$ operation in column $X$ before.
  • Operation type $3$, Pak Chanek will give two coordinates $(x_1, y_1)$ and $(x_2, y_2)$. Calculate how many squares that are Pak Chanek's territory within the rectangular area formed by the square $(x_1, y_1)$ and square $(x_2, y_2)$.

Help Pak Chanek answer every type $3$ operation!

Input Format

The first line contains an integer $T$ — the number of test cases. Each test case is stated as follows.

The first line consists of three integers $N$, $M$, $Q$ — the size of the island and the number of Pak Chanek's questions respectively.

The next $Q$ lines will be of the following format:

  • If the line is of operation type $1$, it will contain two integers $A$ and $X$. It is guaranteed that $A = 1$.
  • If the line is of operation type $2$, it will contain two integers $A$ and $X$. It is guaranteed that $A = 2$.
  • If the line is of operation type $3$, it will contain five integers $A$, $x_1$, $y_1$, $x_2$, and $y_2$. It is guaranteed that $A = 3$.

Output Format

For each operation type $3$, output an integer representing the answer of operation type $3$.

Sample Input

1
4 5 7
1 2
2 3
3 1 1 3 3
3 1 1 1 1
1 3
2 4
3 1 1 4 5

Sample Output

5
0
14

Constraints

  • $1 \le T \le 3$
  • $1 \le N, M \le 1\,000\,000\,000$
  • $1 \le Q \le 100\,000$
  • $1 \le x_1, x_2 \le N$
  • $1 \le y_1, y_2 \le M$
