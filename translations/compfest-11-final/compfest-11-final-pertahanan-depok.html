Description

The Chanek Nation can be represented in the cartesian plane. Chanek's Electricity Company (CEC) is located at $(0, 0)$. The cities in Chanek Nation is located in each $(i, j)$ for all $1 \le i \le 10 \, 000$ and $1 \le j \le 10\,000$.

Chanek Nation is currently experiencing total infrastructure failure. There are $N$ important regions that needs electricity as fast as possible. A region is marked with $x_1, y_1, x_2, y_2$, meaning the region contains all cities $(i, j)$ such that $x_1 \le i \le x_2$ and $y_1 \le j \le y_2$.

CEC can run backup electricity to city $(i, j)$ if there are no other cities in the straight line between $(0, 0)$ and $(i, j)$. Now, Pak Chanek wonders how many cities in the important region that can have backup electricity.

Input Format

The first line consists of an integer $T$ — the number of test cases. Each test case will be stated as follows.

The first line contains $N$ integers — the number of important regions.

The next $N$ lines consists of $4$ integers $x_1, y_1, x_2, y_2$, representing an important region.

Output Format

For each test case, output the number of cities in the important region that gets backup electricity.

Sample Input

1
3
1 1 4 3
2 2 5 4
6 1 7 2

Sample Output

16

Constraints

  • $1 \le T \le 10$
  • $1 \le N \le 50$
  • $1 \le x_1 \le x_2 \le 10\,000$ for all important regions
  • $1 \le y_1 \le y_2 \le 10\,000$ for all important regions
