Description

After years of research, Feli finally discovered a new element. She names this element Jolibi.

The Jolibi element has a unique characteristic. Every Jolibi element has a constant speed with a direction that may not be the same for other Jolibi elements. In addition, a Jolibi element can penetrate other elements, including other Jolibi elements.

Feli becomes more well known because Jolibi is known to produce energy easily. For every pair of Jolibi elements that have a maximum distance of R units, they will produce an energy. If a pair of Jolibi elements have a distance of more than R units, then the energy produced will disappear.

To get that energy, Feli made a device called Jolly that can capture all the energy produced by Jolibi. However, Jolly can only be operated once and anytime. After Jolly is turned on, it will directly capture the produced energy and automaticly turns off afterwards.

In the 0th second, Feli observed that there are $$$N$$$ Jolibi elements with a position and direction that can be represented in two dimensions. Feli wants to use Jolly as effective as possible. As Feli's assistant, you are asked to determine the maximum amount of energy that can be captured using Jolly. Help Feli!

Input Format

The first line contains 2 positive integers $$$N$$$ and $$$R$$$ where $$$N$$$ represents the number of Jolibi elements observed by Feli and $$$R$$$ represents the maximum distance such that Jolibi elements can produce energy. The next $$$N$$$ lines containt 4 integers $$$x, y, dx, dy$$$ where pairs x and y represents the initial position of an element and pairs dx and dy represents the direction vector of the element.

Output Format

Output an integer that shows the maximum energy that can be obtained.

Constraints

  • $$$2 \le N \le 1000$$$
  • $$$0 \le R \le 1000$$$
  • $$$-10000 \le x, y, dx, dy \le 10000$$$

Sample Input

3 2
0 0 1 1
2 0 -2 0
3 3 -1 -1

Sample Output

2
