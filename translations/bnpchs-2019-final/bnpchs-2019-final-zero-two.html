Description

Feli is a Computer Science student. One day, her computer is infected with a strange virus. The virus causes Feli's monitor to display a binary string (consisting of only zeroes and ones).

Due to her boredom, she decided to determine whether the binary string in her monitor contains exactly one "00" substring. Once realizing that the possibility of exactly one "00" substring appearing in the whole string being minuscule, she then decided that she could perform the following operation: swap the positions of any two characters within the string.

Feli wants to change the string as fast as possible, such that the resulting string has exactly one "00" substring. Help Feli by telling her the minimum number of operations required to do it.

A substring is a contiguous segment of a string. For example:

  • "01" is a valid substring of "0100" whereas "11", "101", and "000" is not a valid substring of "0100"
  • "000" has $$$3$$$ "0" substrings, $$$2$$$ "00" substrings, and $$$1$$$ "000" substring

Input Format

Input starts with an integer $$$T$$$ — the number of test cases.
Each test case consists of two lines,
The first line contains an integer $$$N$$$ — the length of the binary string,
The next line contains a binary string of length $$$N$$$

Output Format

For each test case, output "Case #$$$X$$$: $$$Y$$$", where $$$X$$$ is the case number (starting from $$$1$$$), and $$$Y$$$ is the minimum number of operations required. If it is impossible to get a string with exactly one "00" substring, output $$$-1$$$.

Constraints

  • $$$1 \le T \le 5$$$.
  • $$$1 \le N \le 2000$$$.

Sample Input

4
6
101000
10
0100000111
4
0000
5
10011

Sample Output

Case #1: 1
Case #2: 2
Case #3: -1
Case #4: 0

Explanation

  • For case $$$1$$$, we can swap characters at positions $$$1$$$ and $$$5$$$
  • For case $$$2$$$, we can swap characters at positions $$$4$$$ and $$$8$$$, followed by positions $$$6$$$ and $$$10$$$
  • For case $$$3$$$, no matter how many operations are performed, it is impossible to get a string with EXACTLY one "00" substring
  • For case $$$4$$$, the string already contains exactly one "00" substring, so there is no need to perform any operation.
