Description

Dokemon Orange is a game that is newly released. The aim of this game is to catch all Dokemons in the game. Feli, who recently played Dokemon Orange, wants to catch all Dokemons. However, a specific device is needed to catch a Dokemon, called a Dokeball.

Feli recieved information that Dokeballs are actually sold in a special market. This market is special as every player can only do one transaction. The Dokeballs in the special market are sold with the following rules:

  • The price of a Dokeball is an integer between $$$L$$$ and $$$R$$$ inclusive
  • There is only one Dokeball available for every price.
  • In one transaction, the player must choose two integers $$$a$$$ and $$$b$$$ between $$$L$$$ and $$$R$$$ inclusive where $$$a$$$ is less than $$$b$$$
  • The player must buy all Dokeballs from $$$a$$$ to $$$b$$$ inclusive.

The total price to buy all Dokeballs from $$$a$$$ to $$$b$$$ is $$$f(a, b)$$$, where:

$$$ f(a, b) = \left. \begin{cases} \scriptstyle a, & if a = b \\ \scriptstyle g(a, f(a+1, b)), & otherwise \\ \end{cases} \right\} $$$

and

$$$ g(a, b) = a + b - 2 \times (a \wedge b) $$$

where $$$\wedge$$$ represents the bitwise AND operator.

Feli only has $$$M$$$ amount of money and she wants to buy as many Dokeballs as possible. Help Feli so that she can know what is the maximum number of Dokeballs that can be bought!

Input Format

The first line contains a positive integer $$$T$$$ which represents the number of test cases. The next T lines contains 3 positive integers $$$L, R,$$$ and $$$M$$$ where $$$L$$$ until $$$R$$$ represents the price of $$$(R-L+1)$$$ Dokeballs and $$$M$$$ represents the amount of money Feli has.

Output Format

For every test case, the format of output is "Case #$$$X$$$: $$$Y$$$" where $$$X$$$ represents the test case number which is counted from 1 and $$$Y$$$ represents the maximum number of Dokeballs that can be bought by Feli.

Constraints

  • $$$1 \le T \le 100$$$
  • $$$1 \le L \le R \le 10^{18}$$$
  • $$$1 \le (R-L+1) \le 10^5$$$
  • $$$1 \le M \le 10^{18}$$$

Sample Input

2
2 5 100
3 9 9

Sample Output

Case #1: 4
Case #2: 7
