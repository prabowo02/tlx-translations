Description

Fingga Boyz, or so the knights of the kingdom of Bollyjee are called. In the kingdom of Bollyjee, there are $N$ Fingga Boyz that have achieved many victories on the battlefield. Jelix, as king of Bollyjee, wants to give some appreciation to the Fingga Boyz in the form of a house for each person. Due to unknown reasons, there are only $K$ available houses. Each house is numbered from $1$ to $K$. Each member of the Fingga Boyz wants to stay in the house with their family, hence there must be no houses that are given to two or more different knights.

To not cause a racket between the Fingga Boyz, Jelix requests everyone to queue up and give their preferences for their dream home to Jelix personally. In order to be fair, the $i$-th person that arrives only gets $i$ preferences.

After all preferences have been acquired. Jelix starts divvying up the houses. Jelix will start giving the houses (based on their preferences) to the person that gave the preference first. If there are more than one possible houses that could be given, Jelix will give the smaller numbered house that hasn't been given to someone else.

Just as Jelix was about to start divvying the houses, the kingdom of Bondol suddenly arrives to talk business with Jelix. Jelix asks you, as his right-hand man, to divvy up the homes. Determine the house number received by each member of the Fingga Boyz.

Input Format

Input starts with an integer $T$ ($1 \le T \le 10$) — the number of test cases. Each test case starts with two integers $N$ and $K$ ($1 \le N \le K \le 100$) — the member count of the Fingga Boyz and the number of houses available. It is then followed by $N$ lines, where the $i$-th line consists of $i$ distinct integers $P_{i,x}$ ($1 \le P_{i,x} \le K$), representing $i$-th person's home preferences.

Output Format

For each test case, output Kasus #X: , where $X$ is the test case number (starting from $1$). It is then followed by $N$ space-separated integers that show what house number the $i$-th person received. Output Tidak ada pembagian if not everyone can get their own homes.

Sample Input

2
2 3
1
2 3
3 3
1
2 3
1 2 3

Sample Output

Kasus #1: 1 2
Kasus #2: 1 2 3
