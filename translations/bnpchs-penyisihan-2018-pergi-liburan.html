Description

The holiday season has arrived, everyone is going on a holiday trip, including Jelix Fingga. Jelix Fingga, who lives in city $$$1$$$, decided to take a trip with his family to city $$$N$$$ using a car. There are $$$M$$$ bidirectional roads that connect the cities.

On each road, there are landscapes with beauty value $$$W$$$. However, watching the same landscapes will surely be boring. Therefore, the total beauty value is the bitwise operator OR of each road traversed by Jelix Fingga until he reaches his destination.

Formally, the total beauty value is defined as the following :

$$$ {\Large \textrm{Beauty Value} = W_1 \cup W_2 \cup W_3 \dots} $$$

Where $$$W_i$$$ refers to the beauty value of a road that is traversed by Jelix Fingga.

For this trip, Jelix Fingga wants to have a memorable experience. Therefore, he wants the total beauty value when he reaches city $$$N$$$ to be $$$X$$$. However, he doesn't know if that total beauty value can be reached or not. Because he is busy preparing for the trip, he asked you, his wonderful friend, whether it is possible to reach city $$$N$$$ with the total beauty value $$$X$$$.

Input Format

Input starts with an integer $$$T$$$ ($$$1 \le T \le 10$$$) — the number of test cases.

Each test case starts with three integers $$$N$$$, $$$M$$$, and $$$X$$$. ($$$1 \le N \le 1000, 1 \le M \le 20000, 1 \le X \le 10^6$$$) — denoting the number of cities, the number of roads, and the required total beauty value respectively.

Each of the next $$$M$$$ lines contains three integers $$$U$$$, $$$V$$$, and $$$W$$$ ($$$1 \le U, V \le N, 1 \le W \le 10^6$$$) — indicating a road between city $$$U$$$ and city $$$V$$$ with beauty value $$$W$$$

Output Format

For each test case, output "Kasus #$$$X$$$: $$$Y$$$" (without quotes), where $$$X$$$ is the case number (starting from $$$1$$$), and $$$Y$$$ is either "Bisa" if it is possible to reach city $$$N$$$ with beauty value $$$X$$$ or "Tidak" if otherwise.

Sample Input

2
4 5 13
1 2 4
2 4 12
1 4 3
4 3 1
4 3 12
3 3 5
1 2 3
2 3 4
3 1 3

Sample Output

Kasus #1: Bisa
Kasus #2: Tidak

Explanation

For case $$$1$$$, one of the possible routes is traversing through roads $$$1 - 2 - 4 - 4$$$.
