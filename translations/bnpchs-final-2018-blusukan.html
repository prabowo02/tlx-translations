Description

Bollyjee presidential election is around the corner. Jelix as the presidential candidate wants to realize the national commitments by doing blusukan (blusukan is indonesian word that refer to hands-on interaction with public in order to notice their necessity).

There are a bunch of islands and $N$ province in Bollyjee. The island of two provinces lie on could be the same or different. Jelix had $M$ information, each information provide two provinces that lie on same island. If province $A$ and $B$ lie on the same island and province $B$ and $C$ lie on the same island then province $A$ and $C$ also lie on the same island.

Each pair of province in the island is connected by a road. Also for each pair of road there are pavement that connect them so that Jelix can go from a road to other road with this pavement. Furthermore Jelix can use aerial transportation between islands. He can take off from any island and land on any island.

The program that Jelix uses for blusukan is as follow:

  • Jelix can start and end his program in any province on any island.
  • Jelix wants to visit all roads in Bollyjee, where he visits every road exactly once.
  • Pavements and roads are different, a visit in pavement is unnecessary and the order of visit is ignored.
  • Because air transports are expensive, Jelix only fly from a island to other island if he has visited all roads in the island.

Jelix requested you to find how many different combination of his road visit, print the value modulo $10^9 + 7$.

Input Format

Input starts with an integer $T (1 \leq T \leq 100)$ — the number of test cases.
Each test case consists of two integers $N (1 ≤ N ≤ 2000)$ and $M (0 ≤ M ≤ min(10, N)/2))$,
Followed by $M$ lines each contains 2 integers $A$ and $B (1 ≤ A < B ≤ N)$ — province $A$ and $B$ lie on the same island.

Output Format

For each test case, output Kasus #X: Y, where $X$ is the case number (starting from $1$), and $Y$ is the number of different combination of his road visit modulo $10^9 + 7$.

Sample Input

4
10 1
1 2
3 3
1 2
2 3
1 3
5 0
7 4
1 2
2 3
4 5
4 6

Sample Output

Kasus #1: 1
Kasus #2: 6
Kasus #3: 1
Kasus #4: 72

Explanation

In the first case, there is only one road exist thus there is only 1 combination possible.
In the second case, all possible combinations is as follow:

  • (1, 2) → (1, 3) → (2, 3)
  • (1, 2) → (2, 3) → (1, 3)
  • (1, 3) → (1, 2) → (2, 3)
  • (1, 3) → (3, 2) → (2, 1)
  • (2, 3) → (2, 1) → (1, 3)
  • (2, 3) → (3, 1) → (1, 2)
In the third case, Jelix cannot do any road visit because there is no road in the third case, therefore the answer is 1.
In the fourth case, province $(1, 2, 3)$ lie on the same island while on the other island lie province $(4, 5, 6)$ at once.
