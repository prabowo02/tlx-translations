Description

There are four balls. Three of the balls have the same radius and are arranged tightly to form a triangle on a flat plane. The fourth ball has a different radius from the other balls and is placed on top of the arrangement of the 3 balls. Determine the height of the final arrangement of the 4 balls.

Input Format

The first line contains a positive integer $$$T$$$ which represents the number of test cases. The next $$$T$$$ lines contain two positive integers $$$r$$$ and $$$R$$$ where $$$r$$$ represents the radius of the first three balls and $$$R$$$ represents the radius of the fourth ball.

Output Format

For every test case, the format of the output is "Case #$$$X$$$: $$$Y$$$" where $$$X$$$ represents the test case number which is counted from 1, and $$$Y$$$ represents the answer according to the description. The answer will be considered correct if the difference with the judge's answer does not exceed 10-4.

Constraints

  • $$$1 \le T \le 100$$$
  • $$$1 \le r, R \le 100000$$$
  • It is guaranteed that the fourth ball is lifted by the first three balls.

Sample Input

3
5 9
5 4
8 6

Sample Output

Case #1: 26.754084
Case #2: 15.904105
Case #3: 24.519823
