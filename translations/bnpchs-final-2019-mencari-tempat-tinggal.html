Description

Feli is a university student looking for a place to stay. The city where Feli lives in is made up of $$$N$$$ intersections connected by $$$N - 1$$$ roads. It is guaranteed that every pair of intersections are connected directly or indirectly through multiple roads.

Feli, an active university student, often visits $$$M$$$ intersections known as meeting spots. Feli surely wants to stay in a place that minimizes the furthest distance from that place to any of the $$$M$$$ spots. The distance traveled by Feli is the number of intersections that are passed.

Luckily, each intersection has a place that Feli could stay in. Feli can only choose one place to stay in. Help Feli determine where she will stay.

Input Format

Input starts with an integer $$$T$$$ â€” the number of test cases.
Each test case starts with two integers $$$N$$$ and $$$M$$$, where $$$N$$$ is the number of intersections and $$$M$$$ is the number of meeting spots.
Each of the next $$$N - 1$$$ lines contains two integers $$$U$$$ and $$$V$$$, indicating a road between intersection $$$U$$$ and intersection $$$V$$$.
The final line contains $$$M$$$ integers, where each integer denotes one of Feli's meeting spots.

Output Format

For each test case, output "Case #$$$X$$$: $$$Y$$$ $$$Z$$$", where $$$X$$$ is the case number (starting from $$$1$$$), $$$Y$$$ is the intersection number where Feli stays, and $$$Z$$$ is the furthest distance from Feli's location to any of the $$$M$$$ spots.

If there are multiple intersections that Feli could stay in, she would pick the lowest possible intersection number.

Constraints

  • $$$1 \le T \le 5$$$.
  • $$$1 \le N \le 2 \times 10^5$$$.
  • $$$2 \le M \le N$$$.
  • $$$1 \le U, V \le N$$$.

Explanation

For case $$$3$$$, there are two intersections that Feli could stay in, intersection $$$2$$$ or intersection $$$4$$$. She would pick intersection $$$2$$$ since that is the lowest possible intersection number.
