Description

Jelix is a fan of a beverage called es kepal moli. Right now, he has $$$3N$$$ es kepal moli arranged from left to right. Each es kepal moli has a different level of tastiness. Let the level of tastiness be represented as $$$a_1, a_2, ..., a_{3N}$$$.

Because he is generous, Jelix wants to share those es kepal moli to:

1. Beggar
First, Jelix picks any $$$N$$$ es kepal moli to be given to a beggar.

2. Fingga
Then, Jelix picks $$$N$$$ right-most es kepal moli (that has not been given to the beggar) to be shared with his friend Fingga.

3. Jelix
After that, $$$N$$$ remaining es kepal moli are kept for himself.

Jelix wants the value of the sum of level of tastiness of his es kepal moli subtracted by that of Jingga's to be MAXIMIZED. Determine the maximal value that Jelix can obtain.

Input Format

Input starts with a positive integer $$$T$$$ ($$$1 ≤ T ≤ 100$$$) that determines the number of test cases. Each test case consists of two lines.

The first line consists of one positive integer $$$N$$$ ($$$1 ≤ N ≤ 100000$$$).

The next line consists of $$$3N$$$ positive integers separated by spaces which determine $$$a_1, a_2, ..., a_{3N}$$$ ($$$1 ≤ a_i ≤ 10^9$$$).

Output Format

For each test case, output "Kasus #$$$X$$$: $$$Y$$$". $$$X$$$ represents the case number (starting from 1), and $$$Y$$$ represents the maximum difference between Jelix's sum of level of tastiness and Fingga's sum of tastiness.

Sample Input

2
3
8 2 2 7 4 6 5 3 8
1
1 2 3

Sample Output

Kasus #1: 5
Kasus #2: -1

Explanation

For the first test case, Jelix shares his es kepal moli as follows:

  • Gives 2, 2, and 8 es kepal moli to the beggar,
  • Shares 6, 5, and 3 es kepal moli to Fingga,
  • Takes 8, 7, and 4 es kepal moli for himself.

Hence, the difference is (8+7+4) - (6+5+3) = 5.
