Description

Chairoikoguma (otherwise known as Koguma) is a bear that loves honey very much. One time in the honey forest, while wandering around peacefully, Koguma found the location of the tastiest honey in the world. Koguma wants to stay as long as possible while eating the honey.

Unfavorably, Jelix, the owner of the honey forest, despises that his honey is taken. The forest is populated by bees. As soon as Jelix notices Koguma approaching the honey, he sends the bees from the hives to strike Koguma. Koguma wants to eat honey as long as possible, but he hates bee sting. Therefore after Koguma eats the honey for maximum possible time, he will go home safely.

The honey forest can be illustrated as a $N \times N$ meters grid. There are symbols as follow:

  • The symbol K represents the initial position of Koguma,
  • The symbol . represents an empty field,
  • The symbol # represents a woods that can be passed by neither Koguma nor the bees,
  • The symbol @ represents Koguma's house,
  • The symbol H represents a beehive. There can be more than one beehives in the forest.

Koguma's Movements

In one second, Koguma can move at most $S$ steps in four directions: left, right, up, or down. These steps do not necessarily have the same direction. For example if $S = 3$, then in one second Koguma can move with these direction: left-right-up, left-up, up, right-right-right, and so on.

Koguma cannot pass a woods (#), and he can immediately get into his house as soon as he reaches his house (@). Furthermore, Koguma cannot pass a field that contains bee colony (further explanation in the next section) because he will get stung immediately.

Bees Movement

Bees move as a colony, each colony's movement is as follow:

  • In one second, each colony can split itself into at most four new colonies that move into the field to the left, right, up, or down of its current field. The field, where the colony was in, is still populated by the old colony. They remain in place but at the same time move to four directions. Bees only move in the integer second.
  • Formally, if a colony resides in field $(r, c)$ at second $t$, then from second $t$ until $t + 0.999 \dots$ the colony will stay in field $(r, c)$. At second $t + 1$, the fields $(r, c)$, $(r, c + 1)$, $(r, c - 1)$, $(r + 1, c)$, and $(r - 1, c)$ will be populated by bee colonies.
  • The bee colony can pass neither Koguma's house nor a woods. The bees consider Koguma's house as a tree.

The Task

You have been requested to help Koguma specify the maximum time (in seconds) he can enjoy the honey in his initial position and goes home safely without any bee sting.

Input Format

Input starts with a single integer $T (1 \leq T \leq 100)$, where $T$ is the number of test cases.
Each test case begins with 2 integers $N$ $(1 \leq N \leq 800)$ and $S$ $(1 \leq S \leq 1000)$.
The next $N$ lines contain $N$ characters that represent the grid of honey forest.
It is guaranteed that there is exactly one character K and one character @ in the grid, and every bee can reach Koguma's starting position.

Output Format

For each test case, output Kasus #$X$: $Y$, where $X$ is the case number starting from 1, and $Y$ is an integer representing the maximum time (in seconds) Koguma can enjoy the honey. If it is impossible for Koguma to reach his house, output -1 instead.

......#
....##.
...H...

Sample Output

Kasus #1: 2
Kasus #2: 0
Kasus #3: -1
