Description

Chaneka, child of Pak Chanek, is currently doing some math homework. On that homework, each question is of the form "What is the result of $\frac{A \times B}{11}$?" for some given integers $A$ and $B$. It is guaranteed from the input that $A \times B$ is divisible by $11$.

This homework is clearly an easy one. In 10 minutes, Chaneka was able to finish it. However, Chaneka is often hasty. Because of that, to verify Chaneka's answers, Chaneka asks you to create a program that could output the result of $\frac{A \times B}{11}$. Please help Chaneka!

Input Format

The input consists of the two integers $A$ and $B$.

Output Format

An integer, the answer to Chaneka's question.

Sample Input

33 5

Sample Output

15

Constraints

For all subtasks:

  • $1 \le A, B \le 10^{10}$
  • $A \times B$ is divisible by $11$.

Subtask 1 (12 points)

  • $1 \le A, B \le 150$

Subtask 2 (13 points)

  • $1 \le A, B \le 10\,000$

Subtask 3 (20 points)

  • $1 \le A, B \le 10^9$

Subtask 4 (55 points)

  • No additional constraints.
