Description

Jelix has two dice each with $$$N$$$ sides, he has an idea for a game with those two dice. On the die, there are distinct numbers between $$$1$$$ and $$$N$$$ (inclusive) for each side. This game is played by 2 players, player one picks a number between $$$1$$$ and $$$(N - 1)$$$ (inclusive). Then, both players toss their die, if the maximum number that appears on the top of both dice is smaller or equal to the number picked by the first player, then the first player wins the game. If not, then the second player wins.

After creating those rules, Jelix feels that the first player will have a better chance of winning. So, Jelix will always play as the first player.

This time Jelix challenges Fing to play with him. Because Jelix and Fing are best friends, Jelix does not want Fing to feel cheated by picking a number that is too large and resulting in Fing's winning chance being very low. Jelix decided to pick a number such that Jelix's winning rate is higher than Fing's but the difference between their winning rate must be as small as possible. For example, Jelix's winning rate is 50.05% and Fing's winning rate is 49.95%. Help Jelix decide which number he should pick.

Input Format

First line consists of an integer $$$T$$$ which denotes the number of cases. The next $$$T$$$ lines consists of an integer $$$N$$$ according to the description.

Output Format

For every case, output "Kasus #$$$X$$$: $$$Y$$$" without quotation marks where $$$X$$$ denotes the case number starting from one and $$$Y$$$ denotes the number that Jelix has to choose. If it is impossible, then $$$Y$$$ equals to "-1".

Constraints

  • $$$1 ≤ T ≤ 100$$$.
  • $$$2 ≤ N ≤ 10^{9}$$$.

Sample Input 1

2
6
12

Sample Output 1

Kasus #1: 5
Kasus #2: 9
