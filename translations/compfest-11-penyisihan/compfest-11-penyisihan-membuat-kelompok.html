Description

Mr. Chanek is a teacher in a primary school. Mr. Chanek has $2N$ students where the $i$-th student has friendship level $A_i$.

Now Mr. Chanek would like to create $N$ groups where each group consists of $2$ students. However, Mr. Chanek has a problem since the $i$-th student only wanted to group with $j$-th student if their friendship level difference is less than or equal to $K$. Formally, $|A_i - A_j| \le K$.

Now Mr. Chanek is wondering, is there a configuration where he can create $N$ groups.

Input Format

The first line consists of two integers $N$ and $K$.

The following line consists of $2N$ integers representing friendship level of Mr. Chanek's students.

Output Format

A line consists of "Ya" (Yes in Indonesian) if Mr. Chanek is able to create N groups or "Tidak" (No in Indonesian) if not.

Sample Input 1 

3 2
3 1 5 7 6 4

Sample Output 1 

Ya

Sample Input 2 

3 1
3 1 5 7 6 4

Sample Output 2 

Tidak

Explanation

For the first sample, we can create 3 groups, i.e. {3, 1}, {5, 7}, {6, 4}.

For the second sample, we cannot create 3 groups.

Constraints

  • $1 \le N \le 100\,000$
  • $0 \le K \le 100\,000$
  • $1 \le A_i \le 100\,000$ for $1 \le i \le 2N$
